# Procédure gestion des paiements anticipés

> Référentiel SMQ 2025 — version opérationnelle.

## 🎯 Objectif
- Sécuriser l’enregistrement des paiements anticipés.

## 🔄 Processus

1. Vérifier montant et période couverte.
2. Enregistrer la transaction correctement.
3. Émettre un justificatif client.
4. Contrôler l’impact dans le suivi de facturation.

## 🧾 Traçabilité minimale

- Date/heure de traitement
- Canal utilisé (accueil, téléphone, digital)
- Interlocuteur (prospect/adhérent)
- Action suivante planifiée

## ✅ Checklist

- [ ] Processus appliqué
- [ ] Traçabilité réalisée
- [ ] Validation finale effectuée
