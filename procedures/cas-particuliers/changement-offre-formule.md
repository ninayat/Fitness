# Procédure de changement d'offre / formule

> SMQ 2025 — LA VENTE | Mise à jour 09/2024

---

## Règles encadrant le changement

1. Le client **sans engagement** peut effectuer un changement d'offre vers un abonnement avec engagement.
2. Pour les **abonnements spéciaux** (ex : étudiants), le changement est possible **une fois par an**, avec justificatif et réengagement sur 12 mois.
3. Le changement doit s'effectuer **l'avant-dernier jour du mois** (ex : ancien abonnement clôturé le 29 mars, nouveau démarrant le 30 mars).
4. ⚠️ Indiquer à l'adhérent que le changement d'offre **modifie son mode de prélèvement** → passage en prélèvement 4 semaines obligatoire.
5. Le client **avec engagement** ayant bénéficié d'une **offre promotionnelle** peut effectuer un changement. Si la demande intervient avant 6 mois, l'adhérent devra **régulariser la différence**.

---

## Exemple de régularisation

> L'adhérent a souscrit avec FA offerts + 4 semaines offertes. Au bout de 4 mois, il souhaite passer à l'offre étudiant. Il faut calculer le coût sur 4 mois avec la promotion vs l'offre étudiant → l'adhérent règle la différence.

## Procédure QUI / QUOI / COMMENT

| Rôle | Action | Comment |
|------|--------|---------|
| **Client** | Demande de l'adhérent | |
| **Commercial** | Prise en compte + demande de justificatif si nécessaire | |
| **Commercial** | 1. Résilier l'abonnement en cours la veille de l'avant-dernier jour du mois | Sur Résamania |
| **Commercial** | 2. Faire débuter le nouveau contrat l'avant-dernier jour du mois + joindre le justificatif | Sur Résamania |
| **Commercial** | 3. Détailler les modifications de l'abonnement en note | Sur Résamania |
| **Responsable de club** | Vérification du lendemain (ou correction si besoin) | |

## ✅ Checklist

- [ ] Éligibilité vérifiée (conditions d'offre, ancienneté)
- [ ] Justificatif demandé si nécessaire
- [ ] Ancien abonnement résilié à la bonne date
- [ ] Nouveau contrat créé à l'avant-dernier jour du mois
- [ ] Note détaillée dans Résamania
- [ ] Client informé du changement de mode de prélèvement
