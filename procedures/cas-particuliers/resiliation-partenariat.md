# Procédure résiliation contrat partenariat

> Référentiel SMQ 2025 — version opérationnelle.

## 🎯 Objectif
- Gérer proprement la fin d’un partenariat.

## 🔄 Processus

1. Vérifier clauses et préavis contractuels.
2. Notifier selon le canal requis.
3. Stopper les avantages liés à date d’effet.
4. Archiver les éléments contractuels.

## 🧾 Traçabilité minimale

- Date/heure de traitement
- Canal utilisé (accueil, téléphone, digital)
- Interlocuteur (prospect/adhérent)
- Action suivante planifiée

## ✅ Checklist

- [ ] Processus appliqué
- [ ] Traçabilité réalisée
- [ ] Validation finale effectuée
