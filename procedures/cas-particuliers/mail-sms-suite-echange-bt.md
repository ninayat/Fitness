# Mail/SMS suite échange téléphonique BT

> Référentiel SMQ 2025 — version opérationnelle.

## 🎯 Objectif
- Confirmer par écrit les échanges BT.

## 🔄 Processus

1. Envoyer un message de synthèse court.
2. Rappeler la prochaine étape et échéance.
3. Ajouter le contact utile du club.
4. Enregistrer l’envoi dans le CRM.

## 🧾 Traçabilité minimale

- Date/heure de traitement
- Canal utilisé (accueil, téléphone, digital)
- Interlocuteur (prospect/adhérent)
- Action suivante planifiée
