# Procédure Old client

> Référentiel SMQ 2025 — version opérationnelle.

## 🎯 Objectif
- Gérer le retour d’un ancien client avec historique.

## 🔄 Processus

1. Rechercher l’ancien dossier.
2. Contrôler données et éventuels impayés.
3. Proposer offre adaptée au retour.
4. Réactiver le compte proprement.

## 🧾 Traçabilité minimale

- Date/heure de traitement
- Canal utilisé (accueil, téléphone, digital)
- Interlocuteur (prospect/adhérent)
- Action suivante planifiée

## ✅ Checklist

- [ ] Processus appliqué
- [ ] Traçabilité réalisée
- [ ] Validation finale effectuée
