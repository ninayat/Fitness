# Gestion des contremarques locales LDM

> SMQ 2025 — LA VENTE

---

## Principe

En parallèle du service B2B réseau, vous avez la possibilité de **démarcher des entreprises locales** et de leur vendre des contremarques.

---

## Conditions obligatoires

- L'entreprise doit être sur la **zone du club**
- L'entreprise doit être une **unité locale**
- Les contremarques vendues doivent respecter les **tarifs planchers** de la grille tarifaire en vigueur

---

## Procédure de création

### Étape 1 — Bon de commande

La concrétisation s'effectue via un **bon de commande** selon le modèle en vigueur.

### Étape 2 — Création sur Résamania

| Étape | Action |
|-------|--------|
| 1 | Créer la **fiche prospect de l'entreprise** sur Résamania (informations société + adresse de facturation) |
| 2 | Faire la **demande de création d'offre** via le formulaire de demande, en renseignant toutes les informations et en joignant la convention signée ou le bon de commande |

### Étape 3 — Validation

Le **Responsable Régional** vérifie la conformité. Si conforme, les codes de contremarques sont générés sous **8 jours**.

---

## Procédure QUI / QUOI / COMMENT

| Rôle | Action | Comment |
|------|--------|---------|
| **Club** | Création de contremarques | Créer fiche prospect entreprise + faire la demande via formulaire |
| **Responsable Régional** | Validation de l'offre si adéquation avec la grille tarifaire | |
| **Club** | 1/ Transmettre les codes à l'entreprise dès le règlement de la facture | |
| **Club** | 2/ Lors de la venue d'un adhérent avec un code : créer fiche prospect + renseigner le code contremarque | |
| **Club** | 3/ Faire signer le contrat à l'adhérent | |
| **Club** | 4/ Scanner le document justificatif dans la fiche client | |
| **Club** | 5/ Mettre une note dans la fiche client | |

---

## Important

- L'activation des codes génère une **facture du montant total de la commande** sur la fiche de l'entreprise
- Les activations de codes individuels génèrent une **facturation à 0€**
- Si l'offre n'est pas conforme → refaire la demande en respectant les conditions
