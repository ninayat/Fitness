# Procédure et script de vente gestion QR code

> Référentiel SMQ 2025 — version opérationnelle.

## 🎯 Objectif
- Encadrer l’utilisation du QR code dans le parcours vente/activation.

## 🔄 Processus

1. Vérifier que le QR code correspond au bon client.
2. Scanner et contrôler les informations récupérées.
3. Finaliser l’étape commerciale liée.
4. Tracer le traitement dans le système.

## 🧾 Traçabilité minimale

- Date/heure de traitement
- Canal utilisé (accueil, téléphone, digital)
- Interlocuteur (prospect/adhérent)
- Action suivante planifiée

- [ ] Processus appliqué
- [ ] Traçabilité réalisée
- [ ] Validation finale effectuée
