# Procédure de relance commerciale adhérent

> Référentiel SMQ 2025 — version opérationnelle.

## 🎯 Objectif
- Réactiver l’intérêt d’un prospect/adhérent non converti.

## 🔄 Processus

1. Préparer la relance avec historique client.
2. Reprendre le besoin initial.
3. Proposer une prochaine action simple.
4. Planifier un suivi daté.

## 🧾 Traçabilité minimale

- Date/heure de traitement
- Canal utilisé (accueil, téléphone, digital)
- Interlocuteur (prospect/adhérent)
- Action suivante planifiée

## ✅ Checklist

- [ ] Processus appliqué
- [ ] Traçabilité réalisée
- [ ] Validation finale effectuée
