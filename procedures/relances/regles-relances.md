# Règles encadrant les relances

> Référentiel SMQ 2025 — version opérationnelle.

## 🎯 Objectif
- Cadre commun pour des relances efficaces et conformes.

## 🔄 Processus

1. Respecter les consentements de communication.
2. Limiter la pression commerciale excessive.
3. Tracer chaque contact dans le CRM.
4. Varier les canaux (appel, mail, SMS) selon contexte.

## 🧾 Traçabilité minimale

- Date/heure de traitement
- Canal utilisé (accueil, téléphone, digital)
- Interlocuteur (prospect/adhérent)
- Action suivante planifiée

## ✅ Checklist

- [ ] Processus appliqué
- [ ] Traçabilité réalisée
- [ ] Validation finale effectuée
