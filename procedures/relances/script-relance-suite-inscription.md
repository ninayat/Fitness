# Script relance suite inscription

> Référentiel SMQ 2025 — version opérationnelle.

## 🎯 Objectif
- Sécuriser le démarrage après inscription.

## 🔄 Processus

1. Message de bienvenue personnalisé.
2. Rappel des accès et services inclus.
3. Vérification première venue.
4. Proposition d’accompagnement initial.

## 🧾 Traçabilité minimale

- Date/heure de traitement
- Canal utilisé (accueil, téléphone, digital)
- Interlocuteur (prospect/adhérent)
- Action suivante planifiée
