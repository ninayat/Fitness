# Procédure transfert de fiche prospect

> SMQ 2025 — VENTE

---

## Contexte

Lorsque vous renseignez les coordonnées d'un nouveau prospect, sa fiche peut déjà exister dans un autre club. Vous pouvez alors **transférer sa fiche membre** sur votre club sous certaines conditions.

---

## Conditions préalables

- Le prospect **ne doit pas** être déjà inscrit sur votre club (éviter les doublons).
- Le client sur l'autre club doit avoir un statut **prospect** ou **client résilié**.
- Le **numéro de téléphone** doit être à jour pour qu'il reçoive le code de vérification. Si ce n'est pas le cas, appeler l'autre club pour mise à jour.
- Si le client transféré est **en dette**, un mail partira automatiquement pour l'indiquer au club récepteur.

---

## Étapes sur Résamania

| Étape | Action |
|-------|--------|
| 1 | Créer la fiche prospect |
| 2 | Cliquer sur **« Consulter le profil »** |
| 3 | Cliquer sur **« Envoyer le code »** |
| 4 | Saisir le code reçu par le client |
| 5 | Cliquer sur **« Valider »** |

Toutes les coordonnées sont alors transférées sur votre club — vous pouvez poursuivre la vente.

---

> **Note :** Le transfert est un outil commercial pour faciliter le parcours client. Ne pas inciter systématiquement les adhérents à transférer — la libre circulation limite ce besoin.
