# Script de vente téléphonique

> Référentiel SMQ 2025 — version opérationnelle.

## 🎯 Objectif
- Fiabiliser la conversion commerciale par téléphone.

## 🔄 Processus

1. Accroche claire et contextualisée.
2. Découverte (objectif, timing, motivation).
3. Argumentaire court orienté bénéfices.
4. Traitement des objections majeures.
5. Call-to-action ferme (date + heure).

## 🧾 Traçabilité minimale

- Date/heure de traitement
- Canal utilisé (accueil, téléphone, digital)
- Interlocuteur (prospect/adhérent)
- Action suivante planifiée

## ✅ Checklist

- [ ] Processus appliqué
- [ ] Traçabilité réalisée
- [ ] Validation finale effectuée
