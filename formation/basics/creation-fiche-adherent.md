# Création fiche adhérent

> SMQ 2025 — VENTE

---

## Règle de base

Lorsqu'un prospect rentre dans votre club, vous devez **obligatoirement** lui créer une fiche adhérent sur le logiciel **Résamania**. Tous les champs doivent être renseignés.

---

## Informations spécifiques

| Champ | Règle de saisie |
|-------|----------------|
| **Nom / Prénom** | En MAJUSCULES. Remplacer les tirets par un espace. |
| **Adresse mail** | En minuscules. |
| **Adresse postale** | Majuscule aux premières lettres des noms propres. « de, la, l', le, et » en minuscule. En cas de doute : vérifier sur Google Maps. |
| **Code postal** | Choisir dans la liste déroulante obligatoire. |
| **Prescripteur** | Indique le type de prospect (ex : Invité → choisir *Pack Ultimate* dans Source). |
| **Source** | La manière dont il est devenu prospect. |
| **Commercial actif** | Appartient à celui qui saisit le prospect (manuel pour l'instant). |

---

## Communications

Dans l'onglet **Communications**, cocher les **opt-in** pour que l'adhérent reçoive les emails d'information du club.

---

## ✅ Checklist

- [ ] Nom / Prénom en majuscules
- [ ] Email en minuscules
- [ ] Adresse postale correcte
- [ ] Code postal sélectionné dans la liste
- [ ] Prescripteur et Source renseignés
- [ ] Commercial actif renseigné
- [ ] Opt-in communications cochés
