# Script de vente accueil

> Référentiel SMQ 2025 — version opérationnelle.

## 🎯 Objectif
- Uniformiser la prise de parole en face-à-face.

## 🔄 Processus

1. Ouverture : présentation + disponibilité.
2. Questions de découverte (objectifs, fréquence, budget).
3. Reformulation des besoins.
4. Proposition personnalisée.
5. Conclusion avec action immédiate.

## 🧾 Traçabilité minimale

- Date/heure de traitement
- Canal utilisé (accueil, téléphone, digital)
- Interlocuteur (prospect/adhérent)
- Action suivante planifiée

## ✅ Checklist

- [ ] Processus appliqué
- [ ] Traçabilité réalisée
- [ ] Validation finale effectuée
