# Script accueil — Traiter les objections

> Référentiel SMQ 2025 — version opérationnelle.

## 🎯 Objectif
- Répondre aux objections sans rompre la dynamique de vente.

## 🔄 Processus

1. Accueillir l’objection sans contradiction frontale.
2. Reformuler pour valider la compréhension.
3. Apporter une réponse factuelle (valeur, flexibilité, accompagnement).
4. Proposer une alternative (essai, RDV, formule ajustée).

## 🧾 Traçabilité minimale

- Date/heure de traitement
- Canal utilisé (accueil, téléphone, digital)
- Interlocuteur (prospect/adhérent)
- Action suivante planifiée

## ✅ Checklist

- [ ] Processus appliqué
- [ ] Traçabilité réalisée
- [ ] Validation finale effectuée
